Google Data Analytics Capstone Project

Case study: World Bank's world development indicators

Business Task

A fictional non-governmental organization wants to start projects in Myanmar and they are especially interested in health and education developments.

As a junior data analyst, I have been tasked to explore World Development Indicator Data available on Big Query Public Data specific to Myanmar country and make recommendations based on analysis specifically in health and education sectors.

Data Source

Data source is available at bigquery-public-data.world_bank_wdi.

Tools used

The following data analytics tools are used for this project:

  1. SQL to query data
  2. Microsoft Excel for data cleaning, manipulation and initial analysis
  3. Tableau Public for data analysis and visualizations

1. Querying data

SQL script below shows that the dataset contains development indicator data of 265 countries. 

SELECT DISTINCT country_name
FROM `bigquery-public-data.world_bank_wdi.indicators_data`

The following SQL code shows that data is available from 1960 to 2020. 

SELECT min(year),max(year)
FROM `bigquery-public-data.world_bank_wdi.indicators_data`

The following SQL query is run in order to retrieve data specific to Myanmar and saved the file in Excel format. 

SELECT *
FROM `bigquery-public-data.world_bank_wdi.indicators_data`
WHERE country_name='Myanmar'

2. Cleaning and Manipulation of data

Cleaning and manipulation of data are done in Microsoft Excel. There are 1310 unique development indicators for Myanmar.

The steps of cleaning and manipulation involve separating each development indicator into the same category, checking for duplicates, filtering them and copying them into different files and pivoting and transposing data for easy analysis.

Summary

In education, more males than females achieve basic education but more females than male attain tertiary level education.

In health sector, majority of expenses come from out-of-pocket.

Females live longer than males in general. Stunting and wasting are very common in the country with more males than females prone to them.

Death from communicable diseases are declining but death from lifestyle diseases such as cancer, diabetes and heart diseases are increasing especially among men.

Recommendations

Therefore, it is recommended to invest more in female education since female makes up the bigger population and they tend to persevere till tertiary education. It is also recommended to create more job opportunities for women because right now, more male than female are in the labor market with female unemployment being high.

It will also be wise to invest more in male healthcare because they seem to live shorter; male children are more prone to stunting and wasting, and males' mortality from lifestyle diseases are above females'. Moreover, since the government is not doing enough to subsidize individual's healthcare expenses, we can provide programs designed to ease financial burdens on individuals for healthcare.

Further exploration

Since non-communicable (lifestyle) diseases are on the rise, it will be necessary to collect more data on what types of lifestyle diseases are prevalent; what lifestyle factors contribute to the diseases, and then provide programs to alleviate such diseases.
