Google Data Analytics Capstone Project

Case study: World Bank's world development indicators

Business Task

A fictional non-governmental organization wants to start projects in Myanmar and they would like to know which areas are in need of more development in order for them to start investing time and money in those areas. They are especially interested in development in health and education.

As a junior data analyst, I have been tasked to explore World Development Indicator data available on Big Query Public Data specific to Myanmar country and make recommendations based on analysis specifically in health and education sectors.

Data Source

Data source is available at bigquery-public-data.world_bank_wdi.

Tools used

The following data analytics tools are used for this project:

  1. SQL to query data
  2. Microsoft Excel for data cleaning, manipulation and initial analysis
  3. Tableau Public for data analysis and visualizations

1. Querying data

SQL script below shows that the dataset contains development indicator data of 265 countries. 

SELECT DISTINCT country_name
FROM `bigquery-public-data.world_bank_wdi.indicators_data`

The following SQL code shows that data is available from 1960 to 2020. 

SELECT min(year),max(year)
FROM `bigquery-public-data.world_bank_wdi.indicators_data`

The following SQL query is run in order to retrieve data specific to Myanmar and saved the file in Excel format. 

SELECT *
FROM `bigquery-public-data.world_bank_wdi.indicators_data`
WHERE country_name='Myanmar'

2. Cleaning and Manipulation of data

Cleaning and manipulation of data is done in Microsoft Excel. There are 1310 unique development indicators for Myanmar.

The first step of cleaning and manipulation involves separating each development indicator into the same category, filtering them and copying them into different files and pivoting and transposing data for easy analysis.

Summary

In terms of population, more female population than male population with 50% of urban population dwelling in slums.

More males than females achieve basic education but more females than male attain tertiary level education.

Health-wise, in general, females live longer than males. Stunting and wasting are very common in the country with more males than females prone to them.

Death from communicable diseases are declining but death from lifestyle diseases such as cancer, diabetes and heart diseases are increasing.

Recommendations

It is recommended to invest more in female education since female makes up the bigger population and they tend to persevere till tertiary education. And it is also recommended to create more job opportunities for women because right now, more male than female are in the labor market with female unemployment being high.

It will also be wise to invest more in men healthcare since they seem to live shorter and male children are more prone to stunting and wasting.

Deliverables for further exploration

Since non-communicable (lifestyle) diseases are on the rise, it will be necessary to collect more data on what types of lifestyle diseases are prevalent and what lifestyle factors contribute to the diseases and provide programs to alleviate such diseases.
